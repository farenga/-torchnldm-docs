DataLoader class. It allows to load and manage PDEGen (https://github.com/farenga/pdegen/) datasets, or .hdf5 files complying with PDEGen dataset structure.

Args:
config:

(DataLoaderConfig), dataset configuration object.

get_batch(section_id: str, t0_index: int, batch_size_params: int, batch_size_timesteps: int, batch_size_nodes: int | None = None) → tuple[torch.Tensor]

Returns a batch of trajectories (ut), their initial values (u0) the respective parameters instances (mut0_batch), and timesteps (t_batch), sampled from the Snapshots and Parameters tensors (S,P), given batch sizes an initial timestep. In the case batch_size_nodes is passed, the output will be in field form, so the last dimension of the state trajectory tensor will be of shape (batch_size_nodes,n) instead than Nh.

Args:
section_id:

(str), id of the dataset section (‘complete’,’train’,’validation’).

t0_index:

(int), index of the initial timestep (<Nt-batch_size_timesteps)

batch_size_params:

(int), number of sampled trajectories (<=N)

batch_size_timesteps:

(int), trajectories length (<=Nt)

batch_size_nodes:

(int), number of sampled mesh nodes (<=M)

Returns:
u0:

(Tensor), batch of initial values, shape = (batch_size_params, Nh) or (…,batch_size_nodes,n)

mut0:

(Tensor), batch of parameters and timesteps related to the initial values, shape = (batch_size_params, Np)

x:

(Tensor), batch of sampled mesh nodes coordinates, shape = (batch_size, batch_size_nodes, d)

t:

(Tensor), 1d tensor of timesteps, shape = (batch_size_timesteps,)

ut:

(Tensor), batch of state trajectories, shape = (batch_size_timesteps, batch_size_params, Nh) or (…,batch_size_nodes,n)

mut:

(Tensor), batch of parameters and timesteps related to the whole trajectory, shape = (batch_size_timesteps, batch_size, Np)

get_trajectory(section_id: str, t0_index: int, param_index: int, timesteps: int, field_form: bool = False) → tuple[torch.Tensor]

Returns a single trajectory (ut), their initial values (u0) the respective parameters instances (mut0_batch), and timesteps (t_batch), sampled from the Snapshots and Parameters tensors (S,P), given batch sizes an initial timestep. In the case batch_size_nodes is passed, the output will be in field form, so the last dimension of the state trajectory tensor will be of shape (batch_size_nodes,n) instead than Nh.

Args:
section_id:

(str), id of the dataset section (‘complete’,’train’,’validation’).

t0_index:

(int), index of the initial timestep (<Nt-batch_size_timesteps)

param_index:

(int), index of sampled paramter instance (<=N)

timesteps:

(int), trajectories length (<=Nt)

field_form:

(bool), flag enabling field form

Returns:
u0:

(Tensor), initial value, shape = (Nh,) or (M,n)

mut0:

(Tensor), parameters instances and timesteps related to the initial value, shape = (Np)

x:

(Tensor), mesh nodes coordinates, shape = (M, d)

t:

(Tensor), 1d tensor of timesteps, shape = (batch_size_timesteps,)

ut:

(Tensor), state trajectory, shape = (timesteps, Nh) or (…,batch_size_nodes,n)
