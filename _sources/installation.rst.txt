Installation
============

How to install the library
--------------------------------

The library can be installed from the github repository at: https://github.com/farenga/torchnldm. 

In particular, to install from the ``main`` branch:

.. code:: console

    $ pip install git+https://github.com/farenga/torchnldm

While, to install from the ``dev`` branch:

.. code:: console

    $ pip install git+https://github.com/farenga/torchnldm@dev

Otherwise, the complete repository can be cloned and installed as follows:

.. code:: console

    $ git clone https://github.com/farenga/torchnldm
    $ cd torchnldm
    $ pip install .

.. note::
   If the repository is still private, the `GitHub Personal Access Token <https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens>`_ is needed to perform ``pip install``.
   
.. code:: console

    $ pip install git+https://GH_PERSONAL_ACCESS_TOKEN@github.com/farenga/torchnldm
